\chapter{Combinatorial formulas for Macdonald polynomials} \label{chap:models}

\section{Unsigned and Signed Alphabets}

An \vocab{unsigned alphabet} \(\mathcal{A}\) is a totally ordered set.
We write \(\ind{a, b}\) to denote the indicator function of \(a > b\).
Some unsigned alphabets we use are \(\positives = \{1, 2, 3, \ldots\}\)
and its subalphabet \(\interval{n} = \{1, 2, \ldots, n\}\).

A \vocab{signed alphabet} \(\mathcal{A}_\pm\) is a
total order on the disjoint union of two copies of a set \(\mathcal{A}\),
a \vocab{positive} copy \(\mathcal{A}_+\) and a \vocab{negative} copy \(\mathcal{A}_-\).
We write \(\ind{a, b}\) to denote the indicator function of \(a > b\) or \(a = b \in \mathcal{A}_-\).
Given an element \(a \in \mathcal{A}\),
we denote its positive copy by \(a \in \mathcal{A}_+\)
and its negative copy by \(\bar{a} \in \mathcal{A}_-\),
and we say that \(a \in \mathcal{A}\) is
the \vocab{absolute value} of \(a \in \mathcal{A}_+\) and of \(\bar{a} \in \mathcal{A}_-\),
denoted by \(|a| = |\bar{a}| = a\).
Some signed alphabets we use are \(\integers_\pm\), with the total order
\begin{equation}
	1 < \bar{1} < 2 < \bar{2} < 3 < \bar{3} < \cdots,
\end{equation}
and its subalphabet \([n]_\pm\).

For both unsigned and signed alphabets, define
\begin{equation}
	\ind{a, b, c} = \ind{a, b} + \ind{b, c} - \ind{a, c}.
\end{equation}
We have \(\ind{a, b, c} \in \{0, 1\}\)
since it is impossible for \(\ind{a, b} = \ind{b, c} = 1 - \ind{a, c}\).

We will also use the symbol \(\infty\) to denote
an element larger than any element in \(\mathcal{A}\),
so that \(\ind{a, \infty} = 1\) and \(\ind{\infty, a} = 0\) for all \(a \in \mathcal{A}\).
Note that \(\infty\) is not an element of \(\mathcal{A}\).

Here's one way to think about \(\ind{a, b, c}\).
For an unsigned alphabet,
\(\ind{a, b, c}\) indicates whether \(a, b, c\) is cyclically weakly decreasing,
where the earlier of two equal entries is considered smaller than the later one.
That is, in an unsigned alphabet,
\(\ind{a, b, c} = 1\) if and only if
\begin{equation}
	a > b > c \quad \text{or} \quad b > c \geq a \quad \text{or} \quad c \geq a > b.
\end{equation}

In signed alphabets, \(\ind{a, b, c}\) indicates whether \(a, b, c\) is cyclically decreasing, where the earlier of two equal positive entries is considered smaller than the later one, and the earlier of two equal negative entries is considered larger than the later one.

\section{Fillings} \label{sec:fillings}

\subsection{Non-augmented Fillings} \label{subsec:non-augmented-fillings}

The \vocab{(non-augmented) skyline diagram} of a composition
\(\alpha = \composition{\alpha_1, \ldots, \alpha_n}\)
is the set
\begin{equation}
	\dg(\alpha) = \left\{ (i, r) \suchthat i \in \interval{n} \text{ and } r \in \interval{\alpha_i} \right\},
\end{equation}
where the first coordinate indexes the columns and the second coordinate indexes the rows.
In \cref{subsec:augmented-fillings}, we will define the \vocab{augmented skyline diagram} of a composition, hence the optional term \emph{non-augmented} is used when the distinction is necessary.
The elements of \(\dg(\alpha)\) are called \vocab{boxes}.
We draw the box \((i, r)\) as a square in column \(i\) and row \(r\).
\Cref{fig:skyline} provides an example of a skyline diagram.

A box \(u = (i_u, r_u)\) \vocab{is after} \(v = (i_v, r_v)\) in the \vocab{reading order} of \(\dg(\alpha)\) if \(r_u > r_v\), or \(r_u = r_v\) and \(i_u < i_v\).
This defines a total order on the boxes of \(\dg(\alpha)\).

A box \(u = (i_u, r_u)\) \vocab{attacks} \(v = (i_v, r_v)\) in \(\dg(\alpha)\) if
\(r_u = r_v\) and \(i_u < i_v\), that is, \(u\) and \(v\) are in the same row and \(u\) is to the left of \(v\); or
\(r_v = r_u - 1\) and \(i_v < i_u\), that is, \(u\) is in the row directly above \(v\) and \(u\) is to the right of \(v\).
\Cref{fig:attacking} illustrates the two configurations in which a box \(u\) attacks a box \(v\).
Let \(\Attack_{\dg(\alpha)}(u) \subset \dg(\alpha)\) be the set of boxes attacked by \(u\).
When there is no ambiguity, we may simply write \(\Attack(u)\).

Let \(u = (i, r) \in \dg(\alpha)\).
The \vocab{leg set} of \(u\) is the set \(\Leg_{\dg(\alpha)}(u) \subset \dg(\alpha)\) of boxes above \(u\) in the same column, that is, \(\Leg_{\dg(\alpha)}(u) = \left\{ (i, r') \in \dg(\alpha) \suchthat r' > r \right\}\).
The \vocab{leg statistic} of \(u\) is defined as \(\leg_{\dg(\alpha)}(u) = \left|\Leg_{\dg(\alpha)}(u)\right| = \alpha_i - r\).
The \vocab{arm set} of \(u\) is the set \(\Arm_{\dg(\alpha)}(u) \subset \dg(\alpha)\) of boxes \(v\) attacked by \(u\) such that \(\leg_{\dg(\alpha)}(v) \leq \leg_{\dg(\alpha)}(u)\), that is,
\begin{equation}
	\Arm_{\dg(\alpha)}(u) = \left\{ v \in \Attack_{\dg(\alpha)}(u) \suchthat \leg_{\dg(\alpha)}(v) \leq \leg_{\dg(\alpha)}(u) \right\}.
\end{equation}
The \vocab{arm statistic} of \(u\) is defined as \(\arm_{\dg(\alpha)}(u) = \left|\Arm_{\dg(\alpha)}(u)\right|\).
When there is no ambiguity, we may simply write \(\Leg(u)\), \(\leg(u)\), \(\Arm(u)\), and \(\arm(u)\).
The box \vocab{south} of \(u = (i, r)\) is the box \(\south{u} = (i, r-1)\), that is, the box directly below \(u\).
If \(r > 1\), then \(\south{u} \in \dg(\alpha)\); while if \(r = 1\), then \(\south{u}\) is still defined (in row \(0\)),
but it is not in \(\dg(\alpha)\).
See \cref{fig:leg-arm} for an example of the leg set, arm set, and south of a box.

A \vocab{(non-augmented) filling} of shape \(\alpha = \composition{\alpha_1, \ldots, \alpha_n}\) with labels in an alphabet \(\mathcal{A}\) is a function \(T \colon \dg(\alpha) \to \mathcal{A}\).
Let \(\Fill[\mathcal{A}](\alpha)\) denote the set of (non-augmented) fillings of shape \(\alpha\) with labels in \(\mathcal{A}\).

For a signed alphabet \(\mathcal{A}_\pm\) and a filling \(T \in \Fill[\mathcal{A}_\pm](\alpha)\), the \vocab{absolute value of \(T\)} is the filling \(|T| \in \Fill[\mathcal{A}](\alpha)\) defined by \(|T|(u) = |T(u)|\) for all \(u \in \dg(\alpha)\).

For an unsigned alphabet \(\mathcal{A}\),
a filling \(T\in \Fill[\mathcal{A}](\alpha)\)
is \vocab{non-attacking} if \(T(u) \neq T(v)\) whenever \(u\) attacks \(v\) in \(\dg(\alpha)\).
For a signed alphabet \(\mathcal{A}_\pm\),
a filling \(T\in \Fill[\mathcal{A}_\pm](\alpha)\) is \vocab{non-attacking} if \(|T|\) is a non-attacking filling in \(\Fill[\mathcal{A}](\alpha)\).
Let \(\NAF[\mathcal{A}](\alpha) \subseteq \Fill[\mathcal{A}](\alpha)\) denote the set of non-attacking fillings of shape \(\alpha\) with labels in \(\mathcal{A}\).
See \Cref{fig:non-attacking-filling} for an example of a non-attacking filling.

The \vocab{content} of a filling \(T\) with labels in an alphabet \(\mathcal{A}\) is the multiset \(\content(T)\) whose elements are the labels \(a \in \mathcal{A}\), with multiplicity equal to the number of boxes \(u \in \dg(\alpha)\) such that \(T(u) = a\).

When the alphabet is \(\positives\) or \(\interval{n}\),
the content can be identified with the composition \(\beta = \content(T)\) where \(\beta_i\) is the multiplicity of \(i\) in \(\content(T)\).
We define
\[
	\mathbf{x}^T = \mathbf{x}^{\content(T)} = \prod_{i} x_i^{\beta_i}.
\]
For example, the filling \(T\) in \cref{fig:non-attacking-filling} has content \(\content(T) = \composition{2, 1, 1, 2, 3}\) and \(\mathbf{x}^T = \mathbf{x}^{\content(T)} = x_1^2 x_2^1 x_3^1 x_4^2 x_5^3\).

Recall that boxes in row \(0\), which are not in \(\dg(\alpha)\),
can be obtained as the south of boxes in row \(1\).
By convention, if \(w\) is a box in row \(0\),
then we define \(T(w) = \infty\).

Given a filling \(T \in \Fill(\alpha)\),
we say that a box \(u \in \dg(\alpha)\)
is a \vocab{descent} in \(T\) if \(\ind{T(u), T(\south{u})} = 1\).
Note that boxes in row \(1\) are never descents,
since the label of the south of a box in row \(1\) is always \(\infty\), by convention.
Let \(\Des(T) \subseteq \dg(\alpha)\) denote the set of descents in \(T\).
The \vocab{major index} of \(T\) is
\begin{equation}
	\maj(T)
	= \sum_{u \in \Des(T)} \left( \leg(u) + 1 \right)
	= \sum_{u \in \dg(\alpha)} \ind{T(u), T(\south{u})} \left( \leg(u) + 1 \right).
\end{equation}
See \cref{fig:major-index} for an example.

A \vocab{triple} is an ordered tuple \((u, v, w)\) of boxes
such that \(u \in \dg(\alpha)\), \(v \in \Arm(u)\), and \(w = \south{u}\).
The total number of triples in \(\dg(\alpha)\) is
\begin{equation}
	\sum_{u \in \dg(\alpha)} \arm(u).
\end{equation}
If \(v\) is to the right of \(u\) in the same row as \(u\),
then the triple is of \vocab{type A},
and if \(v\) is to the left of \(u\) in the row directly below \(u\),
then the triple is of \vocab{type B}.%
\footnote{This is not related to Cartan types \(A_n\) and \(B_n\).
In \cite{HHL08} and \cite{Fer11},
they are called \vocab{type II} and \vocab{type I}, respectively.}
The configurations of the boxes \(u\), \(v\), and \(w\)
when the tuple \((u, v, w)\) is of type A or type B are shown in \cref{fig:triples}.
Note that if \(\alpha\) is a partition, there are no triples of type B.

Note that the definition of a triple \((u, v, w)\) allows for \(u\) to be in row \(1\), and consequently for \(w\) to be in row \(0\).
A triple that has \(u\) in row \(1\) is called a \vocab{degenerate triple}.
A degenerate triple is always of type A, since \(v \in \Arm(u) \subset \dg(\alpha)\), and therefore cannot be in row \(0\).

% We define the \vocab{inversion} and \vocab{coinversion} statistics,
% following \cite[Section~3]{HHL08}.

Given a filling \(T \in \Fill(\alpha)\),
a triple \((u, v, w)\) in \(\dg(\alpha)\) is
an \vocab{inversion triple} if \(\ind{T(u), T(v), T(w)} = 1\),
and is a \vocab{coinversion triple} otherwise.
Let \(\inv(T)\) and \(\coinv(T)\) denote
the number of inversion triples and coinversion triples in \(T\),
respectively.
% We denote by \(\InversionTriples(T)\) and \(\CoinversionTriples(T)\) the sets of inversion triples and coinversion triples in \(T\), respectively, so that \(\inv(T) = |\InversionTriples(T)|\) and \(\coinv(T) = |\CoinversionTriples(T)|\).
% It is straightforward to check that these definitions of inversion triples
% are equivalent to those in \cite{Ale19} and \cite{CMW22}.
We have
\begin{equation} \label{eq:inv-plus-coinv}
	\inv(T) + \coinv(T) = \sum_{u \in \dg(\alpha)} \arm(u),
\end{equation}

	Then
	\begin{equation}

	\end{equation}
\end{proposition}

\subsection{Augmented Fillings} \label{subsec:augmented-fillings}

In the context of augmented fillings,
we either take the unsigned alphabet \(\mathcal{A} = \interval{n}\),
or the signed alphabet \(\mathcal{A}_\pm = [n]_\pm\).

The \vocab{augmented skyline diagram} of a composition
\(\alpha = \composition{\alpha_1, \ldots, \alpha_n}\)
is the set
\begin{equation}
	\adg(\alpha) = \dg(\alpha) \cup \{(i, 0) \suchthat i \in \interval{n}\},
\end{equation}
which is obtained from \(\dg(\alpha)\) by adding an extra row of \(n\) boxes below row \(1\),
called the \vocab{row \(0\)} or the \vocab{basement}.
\Cref{fig:augmented-skyline} provides an example of an augmented skyline diagram.
Augmented skyline diagrams are always be drawn with the basement shaded in gray to distinguish them from non-augmented skyline diagrams.

The attacking relation, leg set, arm set, leg statistic, and arm statistic in \(\adg(\alpha)\) are defined as in \cref{subsec:non-augmented-fillings}, with boxes in row \(0\) treated like any other boxes.
However, since row \(0\) is now part of the diagram, there are subtle differences: for \(u = (i, r) \in \adg(\alpha)\), if \(r \geq 2\), then \(\Attack_{\adg(\alpha)}(u) = \Attack_{\dg(\alpha)}(u)\) and \(\arm_{\adg(\alpha)}(u) = \arm_{\dg(\alpha)}(u)\); but if \(r = 1\), \(\Attack_{\adg(\alpha)}(u)\) and \(\Arm_{\adg(\alpha)}(u)\) may include boxes in row \(0\) not present in the non-augmented case.
Thus, extra attention is needed when computing these sets and statistics for boxes in row \(1\), as they may differ depending on whether \(\dg(\alpha)\) or \(\adg(\alpha)\) is used.
If we need to distinguish between the two settings, we use the subscripts \(\dg(\alpha)\) or \(\adg(\alpha)\) to indicate which diagram is being used.

The definition of the south of a box is slightly different in the augmented setting.
Given a box \(u = (i, r)\), if \(r \geq 1\), the \vocab{south} of \(u\) is the box \(\south{u} = (i, r-1)\), while if \(r = 0\), then \(\south{u}\) is not defined.
Note that the south of a box in row \(1\) of \(\adg(\alpha)\) is also a box in \(\adg(\alpha)\), unlike in \Cref{subsec:non-augmented-fillings} where the south of a box in row \(1\) is not in \(\dg(\alpha)\).
We also remark that the south of a box in row \(0\) is not a box in row \(-1\) outside \(\adg(\alpha)\), but rather is simply undefined.

An \vocab{augmented filling} of shape \(\alpha = \composition{\alpha_1, \ldots, \alpha_n}\), basement \(\sigma = \composition{\sigma_1, \ldots, \sigma_n}\), and labels in an alphabet \(\mathcal{A}\) is a function \(T \colon \adg(\alpha) \to \mathcal{A}\) such that \(T(i, 0) = \sigma_i\) for all \(i \in \interval{n}\).
Let \(\Fill[\mathcal{A}](\alpha, \sigma)\) denote the set of augmented fillings of shape \(\alpha\), basement \(\sigma\), and labels in \(\mathcal{A}\).
The absolute value and the non-attacking condition for augmented fillings are defined as in \cref{subsec:non-augmented-fillings}.
Let \(\NAF[\mathcal{A}](\alpha, \sigma) \subseteq \Fill[\mathcal{A}](\alpha, \sigma)\) denote the set of non-attacking augmented fillings of shape \(\alpha\), basement \(\sigma\), and labels in \(\mathcal{A}\).

See \Cref{fig:non-attacking-augmented-filling} for an example of a non-attacking augmented filling.
Augmented fillings are always drawn with the basement shaded in gray to distinguish them from non-augmented fillings.

Note that the non-attacking condition implies that, for a non-attacking augmented filling with labels in \(\interval{n}\), the entries \(\sigma_1, \ldots, \sigma_n\) in the basement are distinct.
Therefore, the basement \(\sigma\) can be identified with a permutation of \(\interval{n}\).
For (possibly attacking) augmented fillings with labels in \(\interval{n}\) or \(\interval{n}_\pm\), the definition allows for repeated or negative entries in the basement.
However, the applications in this work only require the case of a permutation basement, that is, positive and distinct entries in the basement.

The \vocab{content} of an augmented filling \(T\) with labels in an alphabet \(\mathcal{A}\) is the multiset \(\content(T)\) whose elements are the labels \(a \in \mathcal{A}\), with multiplicity equal to the number of boxes \(u \in \dg(\alpha)\) such that \(T(u) = a\).
Note that boxes in row \(0\) are not counted in the content.
When the alphabet is \(\positives\) or \(\interval{n}\),
the content can be identified with the composition \(\beta = \content(T)\) where \(\beta_i\) is the multiplicity of \(i\) in \(\content(T)\).
We define
\[	\mathbf{x}^T = \mathbf{x}^{\content(T)} = \prod_{i} x_i^{\beta_i}. \]
For example, the augmented filling \(T\) in \cref{fig:non-attacking-augmented-filling} has content \(\content(T) = \composition{2, 1, 1, 2, 3}\) and \(\mathbf{x}^T = \mathbf{x}^{\content(T)} = x_1^2 x_2^1 x_3^1 x_4^2 x_5^3\).

The definitions of descents, and major index for augmented fillings are as in \cref{subsec:non-augmented-fillings}.
Note that boxes in row \(1\) can be descents, since the label of the south of a box in row \(1\) is now given by the basement, rather than being \(\infty\) by convention.
However, boxes in row \(0\) are never descents, since they do not have a south.

A triple \((u, v, w)\) is defined as in \cref{subsec:non-augmented-fillings}.
Since boxes in row \(0\) do not have a south, \(u\) cannot be in row \(0\).
There is no concept of a degenerate triple in augmented fillings.
The definitions of inversion triples, coinversion triples, and the statistics \(\inv(T)\) and \(\coinv(T)\) for augmented fillings are as in \cref{subsec:non-augmented-fillings}.
In the augmented setting, the total number of triples in \(\adg(\alpha)\) is
\begin{equation} \label{eq:total-triples-augmented}
	\sum_{u \in \dg(\alpha)} \arm_{\adg(\alpha)}(u)
\end{equation}
Note that this sum does not include boxes \(u\) in row \(0\).
We remark that this sum may be different from \(\sum_{u \in \dg(\alpha)} \arm_{\dg(\alpha)}(u)\) from \cref{subsec:non-augmented-fillings}, since the arm statistic for boxes in row \(1\) may be different in the two settings.

Similarly to \cref{subsec:non-augmented-fillings},
there is another interpretation of \(\inv(T)\) and \(\coinv(T)\), in terms of inversion pairs.
An \vocab{inversion pair} in an augmented filling \(T\) is a pair \((u, v)\) of boxes with \(u \in \dg(\alpha)\) and \(v \in \adg(\alpha)\) such that \(u\) attacks \(v\) and \(\ind{T(u), T(v)} = 1\).
Let \(\InversionPairs(T)\) denote the set of inversion pairs in \(T\).

\begin{proposition}[{\cite[739]{HHL05}}] \label{prop:inv-via-inversion-pairs-augmented}
	Let \(T \in \Fill(\alpha)\).
	Then
	\begin{equation}
		\inv(T) = \left|\InversionPairs(T)\right| - \sum_{u \in \Des(T)} \arm(u).
	\end{equation}
\end{proposition}

\Cref{prop:inv-via-inversion-pairs-augmented} gives an alternative description of \(\inv(T)\).
Together with \Cref{eq:total-triples-augmented}, we also obtain an alternative description of \(\coinv(T)\) by
\begin{equation}
	\coinv(T) = \sum_{u \in \dg(\alpha)} \arm(u) - \inv(T)

\subsection{Mistaking augmented fillings for non-augmented fillings} \label{subsec:augmented-to-non-augmented}

If we didn't shade the basement in gray,
it would be easy to mistake an augmented filling for a non-augmented filling.
This would lead to mistakes in computing the statistics \(\inv\) and \(\coinv\).
However, the mistakes would be systematic, and we can quantify them.
In a later chapter, we will use this difference create a bijection for non-attacking fillings using a bijection for augmented fillings.

Let \(T \in \Fill[\mathcal{A}](\alpha, \sigma)\) be an augmented filling of shape \(\alpha = \composition{\alpha_1, \ldots, \alpha_n}\), basement \(\sigma = \composition{\sigma_1, \ldots, \sigma_n}\), and labels in an alphabet \(\mathcal{A}\).

It's straightforward to check that \(\maj(T) = \maj(T')\).

However, \(\inv(T)\) and \(\coinv(T)\) may differ from \(\inv(T')\) and \(\coinv(T')\), respectively.
We define new statistics \(\syminv\) and \(\symcoinv\) on augmented fillings by
\begin{equation}
	\syminv(T) = \inv(T')
	\quad \text{and} \quad
	\symcoinv(T) = \coinv(T').
\end{equation}

\begin{proposition}
	Let \(T \in \Fill[\mathcal{A}](\alpha, \sigma)\) be an augmented filling of shape \(\alpha = \composition{\alpha_1, \ldots, \alpha_n}\), basement \(\sigma = \composition{\sigma_1, \ldots, \sigma_n}\).
	Then,
	\begin{equation}
		\syminv(T)
		= \inv(T) +
		\left| \left\{ (i, j) \suchthat 1 \leq i < j \leq n \text{ and } \alpha_i \geq \alpha_j \text{ and } \ind{\sigma_i, \sigma_j} = 1 \right\} \right|
	\end{equation}
	and
	\begin{equation}
		\symcoinv(T)
		= \coinv(T) +
		\left| \left\{ (i, j) \suchthat 1 \leq i < j \leq n \text{ and } \alpha_i \geq \alpha_j \text{ and } \ind{\sigma_i, \sigma_j} = 0 \right\} \right|.
	\end{equation}
\end{proposition}

\begin{proof}
	Consider a triple \((u, v, w)\) in \(\adg(\alpha)\).
	Define \(u' = u + (0, 1)\), \(v' = v + (0, 1)\), and \(w' = w + (0, 1)\).
	Then, \((u', v', w')\) is a triple in \(\dg(\alpha')\).
	Note that all triples \((u', v', w')\) such that \(u'\) is not in row \(1\) are obtained in this way from triples \((u, v, w)\) in \(\adg(\alpha)\).
	Moreover, \((u, v, w)\) is an inversion triple in \(T\) if and only if \((u', v', w')\) is an inversion triple in \(T'\).
	Therefore,
	\begin{equation} \label{eq:inv-prime-not-row-1}
		\left| \{ \text{inversion triples } (u', v', w') \text{ in } T' \suchthat u' \text{ is not in row } 1 \} \right|
		= \inv(T),
	\end{equation}
	and similarly,
	\begin{equation} \label{eq:coinv-prime-not-row-1}
		\left| \{ \text{coinversion triples } (u', v', w') \text{ in } T' \suchthat u' \text{ is not in row } 1 \} \right|
		= \coinv(T).
	\end{equation}

	Now, consider triples \((u', v', w')\) in \(\dg(\alpha')\) such that \(u'\) is in row \(1\).
	This is equivalent to \(u' = (i, 1)\) for some \(i \in \interval{n}\), and \(v' = (j, 1)\) for some \(j \in \interval{n}\) such that \(j > i\) and \(\alpha'_i \geq \alpha'_j\).
	This is an inversion triple in \(T'\) if and only if \(\ind{\sigma_i, \sigma_j, \infty} = 1\), which is equivalent to \(\ind{\sigma_i, \sigma_j} = 1\).
	Therefore,
	\begin{multline} \label{eq:inv-prime-row-1}
		\left| \{ \text{inversion triples } (u', v', w') \text{ in } T' \suchthat u' \text{ is in row } 1 \} \right| = \\
		= \left| \left\{ (i, j) \suchthat 1 \leq i < j \leq n \text{ and } \alpha_i \geq \alpha_j \text{ and } \ind{\sigma_i, \sigma_j} = 1 \right\} \right|,
	\end{multline}
	and similarly,
	\begin{multline} \label{eq:coinv-prime-row-1}
		\left| \{ \text{coinversion triples } (u', v', w') \text{ in } T' \suchthat u' \text{ is in row } 1 \} \right| = \\
		= \left| \left\{{(i, j) \suchthat 1 \leq i < j \leq n \text{ and } \alpha_i \geq \alpha_j \text{ and } \ind{\sigma_i, \sigma_j} = 0} \right\} \right|.
	\end{multline}

	The results follow by adding \Cref{eq:inv-prime-not-row-1} and \Cref{eq:inv-prime-row-1}, and adding \Cref{eq:coinv-prime-not-row-1} and \Cref{eq:coinv-prime-row-1}.
\end{proof}

\subsection{Modified Macdonald polynomials \texorpdfstring{\(\tilde{H}_\mu\)}{H̃μ}} \label{subsec:modified-macdonald}

Recall that the Modified Macdonald polynomials \(\tilde{H}_\mu\) are a family of polynomials indexed by partitions \(\mu\).

In \cite{HHL05}, \citeauthor{HHL05} give a combinatorial formula for \(\tilde{H}_{\mu'}\) as%
\footnote{Here, we use the conjugate partition \(\mu'\) instead of \(\mu\) to match the conventions in \cite{HHL05}. This is because the combinatorial model in \cite{HHL05} is based on Ferrers diagrams, which correspond to the conjugate of the skyline diagrams used in this work.}
\begin{equation} \label{eq:modified-macdonald-Fill}
	\tilde{H}_{\mu'}(\mathbf{x}; q, t) =
	\sum_{T \in \Fill[\positives](\mu)} q^{\inv(T)} t^{\maj(T)} \mathbf{x}^T,
\end{equation}
where \(\Fill[\positives](\mu)\) is the set of fillings of shape \(\mu\) with labels in the positive integers \(\positives\).

In \cite{HHL08}, \citeauthor{HHL08} show that the set may be changed to \(\Fill[\positives](\alpha)\) for any rearrangement \(\alpha\) of \(\mu\), and the formula still holds.
In \Cref{chap:bijective}, we give a bijective proof of this rearrangement property.

Recall that the integral form Macdonald polynomials \(J_\mu\) are a family of polynomials indexed by partitions \(\mu\).
\citeauthor{HHL05} \cite[Section~7]{HHL05} derive a tableaux formula for the integral form Macdonald polynomials \(J_\mu\) from the combinatorial formula for the modified Macdonald polynomials \(\tilde{H}_\mu\) in \Cref{eq:modified-macdonald-Fill}.
We briefly recall the main steps.  

From the plethystic formula in \Cref{eq:modified-macdonald-plethystic}, and using the symmetry \(\tilde{H}_\mu(\mathbf{x}; q, t) = \tilde{H}_{\mu'}(\mathbf{x}; t, q)\), we have
\begin{equation} \label{eq:J-in-terms-of-tilde-H}
	J_\mu(\mathbf{x}; q, t)
	= t^{n(\mu)} \tilde{H}_\mu\left[ X (1 - t^{-1}); q, t^{-1} \right]
	= t^{n(\mu)+|\mu|} \tilde{H}_{\mu'}\left[ X (t^{-1} - 1); t^{-1}, q \right],
\end{equation}
where \(n(\mu) = \sum_{i} (i-1) \mu_i = \sum_{u \in \dg(\mu)} \arm(u)\) and \(|\mu| = \sum_i \mu_i\).
Applying a standard technique called \emph{superization},
\cite{HHL05} derive from \Cref{eq:modified-macdonald-Fill,eq:J-in-terms-of-tilde-H} the following combinatorial formula for \(J_\mu\) as a sum over all fillings using the signed alphabet \(\interval{n}_\pm\):
\begin{equation} \label{eq:J-Fill-signed}
	J_\mu(\mathbf{x}; q, t)
	= \sum_{T \in \Fill[\interval{n}_\pm](\mu)}
	\mathbf{x}^{|T|}
	q^{\maj(T)}
	t^{\coinv(T)}
	(-t)^{\operatorname{neg}(T)},
\end{equation}
where \(\operatorname{neg}(T)\) denotes the number of negative entries in \(T\).  
We refer the reader to \cite[Section~7]{HHL05} for a complete derivation.

Two other combinatorial formulas for \(J_\mu\) in terms of non-attacking fillings are also given in \cite{HHL05}, below in \Cref{prop:J-NAF-signed,prop:J-NAF-unsigned}.
We reproduce their proofs because their ideas are useful in \Cref{sec:integral-form-permuted-basement}.

\begin{proposition} \label{prop:J-NAF-signed}
	For a partition \(\mu\) with at most \(n\) parts,
	the integral form Macdonald polynomial \(J_\mu(\mathbf{x}; q, t)\) can be expressed as a sum one over all non-attacking fillings using the signed alphabet \(\interval{n}_\pm\),
\begin{equation} \label{eq:J-NAF-signed}
	J_\mu(\mathbf{x}; q, t)
	= \sum_{T \in \NAF[\interval{n}_\pm](\mu)}
	\mathbf{x}^{|T|}
	q^{\maj(T)}
	t^{\coinv(T)}
	(-t)^{\operatorname{neg}(T)}.
\end{equation}
\end{proposition}

\begin{proof}[Proof ({\cite[744]{HHL05}})]
	We define a sign-reversing involution on the set of attacking fillings in \(\Fill[\interval{n}_\pm](\mu) \setminus \NAF[\interval{n}_\pm](\mu)\).
	Recall that, in \(\interval{n}_\pm\), we have
	\begin{equation} \label{eq:signed-alphabet-ordering}
		1 < \bar{1} < 2 < \bar{2} < \cdots < n < \bar{n}.
	\end{equation}
	This ordering has two useful properties:
	% First, for any \(a \in \interval{n}\) and any \(x \in \interval{n}_\pm\), we have
	% \begin{equation}
	% 	\ind[,]{a, x} = \ind[,]{\bar{a}, x}.
	% \end{equation}
	% Second, for any \(a \in \interval{n}\) and any \(x \in \interval{n}_\pm\) with \(|x| \neq a\), we have
	% \begin{equation}
	% 	\ind[,]{x, a} = \ind[,]{x, \bar{a}}.
	% \end{equation}
	\begin{align}
		\ind{a, k} = \ind{\bar{a}, k} &\quad \text{for any } a \in \interval{n} \text{ and any } k \in \interval{n}_\pm, \text{ and} \label{eq:involution-property-1} \\
		\ind{k, a} = \ind{k, \bar{a}} &\quad \text{for any } a \in \interval{n} \text{ and any } k \in \interval{n}_\pm \text{ with } |k| \neq a. \label{eq:involution-property-2}
	\end{align}

	Given an attacking filling \(T \in \Fill[\interval{n}_\pm](\mu) \setminus \NAF[\interval{n}_\pm](\mu)\),
	let \(a\) be the smallest label which is the absolute value of two attacking entries in \(T\), which exists since \(T\) is attacking.
	Let \(v\) be the earlier box (in reading order) with entry of absolute value \(a\) that is attacked by another box with entry of absolute value \(a\).
	Let \(u\) be the earlier box (in reading order) which attacks \(v\) and has absolute value \(a\).
	Define \(T'\) to be the filling obtained from \(T\) by switching the sign of the entry in box \(u\).
	Then, \(T'\) is also an attacking filling in \(\Fill[\interval{n}_\pm](\mu) \setminus \NAF[\interval{n}_\pm](\mu)\).

	Since the definitions of \(a\), \(v\) and \(u\) only depend on the absolute values of the entries in \(T\),
	we have that applying the same procedure to \(T'\) returns \(T\).
	Hence, the map \(T \mapsto T'\) is an involution.

	Without loss of generality, assume that \(T(u) = a\) and \(T'(u) = \bar{a}\).
	It is straightforward to check that \(\mathbf{x}^{|T'|} = \mathbf{x}^{|T|}\) and \(\operatorname{neg}(T') = \operatorname{neg}(T) + 1\).

	I claim that \(\Des(T') = \Des(T)\), which implies that \(\maj(T') = \maj(T)\).
	Let \(w\) be the box directly above \(u\), if it exists.
	For any box \(b \notin \{w, u\}\), the fillings \(T\) and \(T'\) agree on the entries in boxes \(b\) and \(\south{b}\),
	so \(b\) is a descent in \(T\) if and only if it is a descent in \(T'\).
	For \(w\), if it exists, we can apply \Cref{eq:involution-property-1} to conclude that \(w\) is a descent in \(T\) if and only if it is a descent in \(T'\).
	For \(u\), note that \( d(u) \) is attacked by \(v\) and is earlier than \(v\) in reading order.
	Therefore, by minimality of \(v\), we have \(|T(d(u))| \neq a\), so we can apply \Cref{eq:involution-property-2} to conclude that \(u\) is a descent in \(T\) if and only if it is a descent in \(T'\).
	Thus, we have shown that \(\Des(T') = \Des(T)\), which implies that \(\maj(T') = \maj(T)\).

	I claim that \(\inv(T') = \inv(T) + 1\).
	Using \Cref{prop:inv-via-inversion-pairs-augmented}, since \(\Des(T') = \Des(T)\), it suffices to show that \(\left|\InversionPairs(T')\right| = \left|\InversionPairs(T)\right| + 1\).
	Any pair of attacking boxes that does not involve box \(u\) is an inversion pair in \(T\) if and only if it is an inversion pair in \(T'\).
	Moreover, by \Cref{eq:involution-property-1}, any box \(b\) that attacks \(u\) forms an inversion pair \((b, u)\) in \(T\) if and only if it forms an inversion pair \((b, u)\) in \(T'\).
	Consider a box \(b\) that is attacked by \(u\).
	If \(b = v\), then \((u, v)\) is an inversion pair in \(T'\) but not in \(T\).
	If \(b \neq v\), then by minimality of \(u\), we have \(|T(b)| \neq a\), so by \Cref{eq:involution-property-2}, \((u, b)\) is an inversion pair in \(T\) if and only if it is an inversion pair in \(T'\).
	Therefore, \(\InversionPairs(T')\) has exactly one more inversion pair than \(\InversionPairs(T)\), namely \((u, v)\),
	which proves the claim.
	Consequently, \(\coinv(T') = \coinv(T) - 1\).

	Putting this together, we have
	\begin{align*}
		\mathbf{x}^{|T'|} q^{\maj(T')} t^{\coinv(T')} (-t)^{\operatorname{neg}(T')}
		&= \mathbf{x}^{|T|} q^{\maj(T)} t^{\coinv(T) - 1} (-t)^{\operatorname{neg}(T) + 1} \\
		&= - \mathbf{x}^{|T|} q^{\maj(T)} t^{\coinv(T)} (-t)^{\operatorname{neg}(T)},
	\end{align*}
	which implies that the contributions of fillings in \(\Fill[\interval{n}_\pm](\mu) \setminus \NAF[\interval{n}_\pm](\mu)\) to the sum in \Cref{eq:J-Fill-signed} cancel out, and the result follows.
\end{proof}

\begin{proposition} \label{prop:J-NAF-unsigned}
	For a partition \(\mu\) with at most \(n\) parts,
	the integral form Macdonald polynomial \(J_\mu(\mathbf{x}; q, t)\) can be expressed as a sum one over all non-attacking fillings using the unsigned alphabet \(\interval{n}\),
	\begin{equation} \label{eq:J-NAF-unsigned}
		J_\mu(\mathbf{x}; q, t)
		= \sum_{T \in \NAF[\interval{n}](\mu)} \mathbf{x}^T
		\mkern -10mu
		\prod_{\substack{u \in \dg(\mu) \\ T(u) = T(\south{u})}}
		\mkern -18mu
		(1 - q^{1 + \leg(u)} t^{1 + \arm(u)})
		\mkern -30mu
		\prod_{\substack{u \in \dg(\mu) \\ T(u) \neq T(\south{u}) \text{ or } u \text{ in row } 1}}
		\mkern -38mu
		(1 - t).
	\end{equation}
\end{proposition}

\begin{proof}[Proof ({\cite[750]{HHL05}})]
	Let \(S \in \NAF[\interval{n}_\pm](\mu)\) be a non-attacking signed filling, and let \(|S| \in \NAF[\interval{n}](\mu)\) be the non-attacking unsigned filling obtained from \(S\) by taking the absolute value of each entry.

	For any triple \((u, v, w)\), since \(u\) attacks \(v\) and \(v\) attacks \(w\), the non-attacking condition forces
	\(|S(u)| \neq |S(v)|\) and \(|S(v)| \neq |S(w)|\).
	Define the \emph{absolute coinversion number} \(\operatorname{acoinv}(S)\) to be the number of coinversion triples \((u, v, w)\) in \(S\) for which \(|S(u) \neq |S(w)|\) (and consequently \(|S(u)|\), \(|S(v)|\), and \(|S(w)|\) are all distinct).
	Define the \emph{signed coinversion number} \(\operatorname{scoinv}(S)\) to be the number of coinversion triples \((u, v, w)\) in \(S\) for which \(|S(u)| = |S(w)|\).
	Then, we have
	\begin{equation}
		\coinv(S) = \operatorname{acoinv}(S) + \operatorname{scoinv}(S).
	\end{equation}
	Note that, since \(\ind{a, b, a} = \ind{a, b} + \ind{b, a} - \ind{a, a} = 1\) for all distinct \(a, b \in \interval{n}_\pm\), we have \(\operatorname{scoinv}(|S|) = 0\) and consequently \(\operatorname{acoinv}(S) = \operatorname{acoinv}(|S|) = \coinv(|S|)\).

	Similarly, define the \emph{absolute major index} \(\operatorname{amaj}(S)\) and the \emph{signed major index} \(\operatorname{smaj}(S)\) by
	\begin{equation}
		\operatorname{amaj}(S) = \sum_{\substack{u \in \Des(S) \\ |S(u)| \neq |S(\south{u})|}} (1 + \leg(u)) 
		\quad \text{and} \quad
		\operatorname{smaj}(S) = \sum_{\substack{u \in \Des(S) \\ |S(u)| = |S(\south{u})|}} (1 + \leg(u)).
	\end{equation}
	Then, we have
	\begin{equation}
		\maj(S) = \operatorname{amaj}(S) + \operatorname{smaj}(S).
	\end{equation}
	Note that, since \(\ind{a, a} = 0\) for all \(a \in \interval{n}_\pm\), we have \(\operatorname{smaj}(|S|) = 0\) and consequently \(\operatorname{amaj}(S) = \operatorname{amaj}(|S|) = \maj(|S|)\).

	We group the fillings \(S \in \NAF[\interval{n}_\pm](\mu)\) by their absolute filling \(|S|\) to rewrite \Cref{eq:J-NAF-signed} as
	\begin{equation}
		J_\mu(\mathbf{x}; q, t)
		= \sum_{T \in \NAF[\interval{n}](\mu)}
		\sum_{\substack{S \in {\NAF[\interval{n}_\pm](\mu)} \\ |S| = T}}
		\mkern -10mu
		\mathbf{x}^{|S|} q^{\maj(S)} t^{\coinv(S)} (-t)^{\operatorname{neg}(S)}.
	\end{equation}
	Fix a non-attacking unsigned filling \(T \in \NAF[\interval{n}](\mu)\).
	Using the decompositions of \(\maj(S)\) and \(\coinv(S)\) above,
	the inner sum corresponding to \(T\) can be rewritten as
	\begin{equation*}
		\sum_{\substack{S \in {\NAF[\interval{n}_\pm](\mu)} \\ |S| = T}}
		\mkern -10mu
		\mathbf{x}^{|S|} q^{\maj(S)} t^{\coinv(S)} (-t)^{\operatorname{neg}(S)}
		= \mathbf{x}^T q^{\operatorname{maj}(T)} t^{\coinv(T)}
		\mkern -10mu
		\sum_{\substack{S \in {\NAF[\interval{n}_\pm](\mu)} \\ |S| = T}}
		\mkern -10mu
		q^{\operatorname{smaj}(S)} t^{\operatorname{scoinv}(S)} (-t)^{\operatorname{neg}(S)}.
	\end{equation*}

	Let \(u \in \dg(\mu)\) such that \(T(u) \neq T(\south{u})\) or \(u\) is in row \(1\).
	Then, in any signed filling \(S\) with \(|S| = T\),
	the box \(u\) does not contribute to \(\operatorname{smaj}(S)\), and any triple \((u, v, w)\) does not contribute to \(\operatorname{scoinv}(S)\).
	The entry in box \(u\) contributes to \(\operatorname{neg}(S)\) if and only if \(u \in \interval{n}_-\).

	Now, let \(u \in \dg(\mu)\) such that \(T(u) = T(\south{u})\),
	and let \(S\) be a signed filling with \(|S| = T\).
	The box \(u\) contributes to \(\operatorname{smaj}(S)\) if and only if \(\ind{S(u), S(\south{u})} = 1\),
	which occurs if and only if \(S(u) \in \interval{n}_-\).
	Moreover, any triple \((u, v, w)\) contributes to \(\operatorname{scoinv}(S)\) if and only if \(\ind{S(u), S(v), S(w)} = 0\),
	which occurs if and only if \(S(u) \in \interval{n}_-\).
	Recall that there are \(\arm(u)\) such triples \((u, v, w)\).
	Finally, the entry in box \(u\) contributes to \(\operatorname{neg}(S)\) if and only if \(S(u) \in \interval{n}_-\).

	Note that all of these contributions only depend on the sign of the entry in box \(u\),
	and are independent of the signs of the entries in other boxes.
	Therefore, we can factor the sum over all \(S\) with \(|S| = T\) as a product over all boxes \(u \in \dg(\mu)\).
	If \(T(u) \neq T(\south{u})\) or \(u\) is in row \(1\), the contribution from box \(u\) is
	\begin{equation}
		1 + (-t) = 1 - t,
	\end{equation}
	while if \(T(u) = T(\south{u})\), the contribution from box \(u\) is
	\begin{equation}
		1 + q^{1 + \leg(u)} t^{\arm(u)} (-t) = 1 - q^{1 + \leg(u)} t^{1 + \arm(u)}.
	\end{equation}
	Therefore, the inner sum corresponding to a fixed \(T \in \NAF[\interval{n}](\mu)\) is
	\begin{equation*}
		\sum_{\substack{S \in {\NAF[\interval{n}_\pm](\mu)} \\ |S| = T}}
		\mkern -10mu
		q^{\operatorname{smaj}(S)} t^{\operatorname{scoinv}(S)} (-t)^{\operatorname{neg}(S)}
		=
		\mkern -18mu
		\prod_{\substack{u \in \dg(\mu) \\ T(u) = T(\south{u})}}
		\mkern -18mu
		(1 - q^{1 + \leg(u)} t^{1 + \arm(u)})
		\mkern -30mu
		\prod_{\substack{u \in \dg(\mu) \\ T(u) \neq T(\south{u}) \text{ or } u \text{ in row } 1}}
		\mkern -38mu
		(1 - t),
	\end{equation*}
	and adding over all \(T \in \NAF[\interval{n}](\mu)\) gives the desired result.
\end{proof}

\subsection{Symmetric Macdonald polynomials \texorpdfstring{\(P_\mu\)}{Pμ}}

Recall that the symmetric Macdonald polynomials \(P_\mu\) are a family of polynomials indexed by partitions \(\mu\).
The integral form Macdonald polynomials \(J_\mu\) is a scalar multiple of \(P_\mu\), with relationship given by
\begin{equation}
	J_\mu(\mathbf{x}; q, t)
	=
	c_\mu(q, t)
	P_\mu(\mathbf{x}; q, t),
\end{equation}
where the normalization factor \(c_\mu(q, t)\) is given in \Cref{eq:normalization-factor-c-mu}.

Therefore, from \Cref{eq:J-NAF-unsigned}, we obtain a tableaux formula for \(P_\mu\) as
\begin{equation} \label{eq:P-NAF-unsigned}
	P_\mu(\mathbf{x}; q, t)
	=
	\frac{d_\mu(q, t)}{c_\mu(q, t)}
	\sum_{T \in \NAF[\interval{n}](\mu)} \mathbf{x}^T
	q^{\maj(T)} t^{\coinv(T)}
	\prod_{\substack{u \in \dg(\mu) \\ T(u) \neq T(\south{u})}}
	\frac{1 - t}{1 - q^{1 + \leg(u)} t^{1 + \arm(u)}},
\end{equation}
where the factor \(d_\mu(q, t)\) is given in \Cref{eq:normalization-factor-d-alpha}.

The normalization factors make this formula less appealing.
See \cite[Section~4]{CHMMW22} for more details and other formulas for \(P_\mu\).

\subsection{Permuted-basement Macdonald polynomials \texorpdfstring{\(E_\alpha^\sigma\)}{Eασ}}
\label{sec:tableau-formula}

Recall that the permuted-basement Macdonald polynomials \(E_\alpha^\sigma\) are a family of polynomials indexed by a composition \(\alpha\) and a permutation \(\sigma\).
\citeauthor{Fer11} \cite[Definition~4.4.2]{Fer11} gives a tableau formula for permuted-basement Macdonald polynomials \(E_\alpha^\sigma\) as
\begin{equation} \label{eq:E-NAF-unsigned}
	E^{\sigma}_{\alpha}(\mathbf{x}; q, t) =
	\sum_{T \in \NAF[\interval{n}](\alpha, \sigma)}
	\mathbf{x}^{T}
	q^{\maj(T)} t^{\coinv(T)}
	\prod_{\substack{u \in \dg(\alpha) \\ T(u) \neq T(\south{u})}}
	\frac{1 - t}{1 - q^{1 + \leg(u)} t^{1 + \arm(u)}}.
\end{equation}

\subsection{Integral form permuted-basement Macdonald polynomials \texorpdfstring{\(\mathcal{E}_\alpha^\sigma\)}{Eασ}} \label{sec:integral-form-permuted-basement}

Recall that the integral form permuted-basement Macdonald polynomials \(\mathcal{E}_\alpha^\sigma\) are defined in \Cref{eq:integral-form-permuted-basement-macdonald-polynomial} by
\begin{equation}
	\mathcal{E}_\alpha^\sigma(\mathbf{x}; q, t)
	=
	\left(
		\prod_{u \in \dg(\alpha)} (1 - q^{1 + \leg(u)} t^{1 + \arm(u)})
	\right)
	E_\alpha^\sigma(\mathbf{x}; q, t).
\end{equation}
Therefore, from \Cref{eq:E-NAF-unsigned}, we obtain a tableaux formula for \(\mathcal{E}_\alpha^\sigma\) as
\begin{equation} \label{eq:calE-NAF-unsigned}
	\mathcal{E}^{\sigma}_{\alpha}(\mathbf{x}; q, t) =
	\mkern 15mu
	\sum_{\mathclap{T \in \NAF[\interval{n}](\alpha, \sigma)}}
	\mkern 15mu
	\mathbf{x}^{T}
	q^{\maj(T)} t^{\coinv(T)}
	\mkern 15mu
	\prod_{\mathclap{\substack{u \in \dg(\alpha) \\ T(u) \neq T(\south{u})}}}
	\mkern 15mu
	(1 - q^{1 + \leg(u)} t^{1 + \arm(u)})
	\mkern 15mu
	\prod_{\mathclap{\substack{u \in \dg(\alpha) \\ T(u) \neq T(\south{u})}}}
	\mkern 15mu
	(1 - t).
\end{equation}

Note the similarity of \Cref{eq:calE-NAF-unsigned} for the integral form permuted-basement Macdonald polynomials \(\mathcal{E}_\alpha^\sigma\)
with \Cref{eq:J-NAF-unsigned} for the integral form symmetric Macdonald polynomials \(J_\mu\), the third of three formulas for \(J_\mu\) given in \Cref{subsec:integral-form-macdonald}.

We can repeat the arguments used in \Cref{prop:J-NAF-unsigned,prop:J-NAF-signed} (in reverse order) to obtain formulas for \(\mathcal{E}_\alpha^\sigma\) that are similar to \Cref{eq:J-NAF-signed}, the second formula for \(J_\mu\), and \Cref{eq:J-Fill-signed}, the first formula for \(J_\mu\).
The arguments are nearly identical, with some minor adjustments to account for the basement \(\sigma\) and to account for the reverse order of the steps.
We nevertheless fully reproduce them below for self-containment.

\begin{proposition} \label{prop:calE-NAF-signed}
	For a composition \(\alpha = \composition{\alpha_1, \ldots, \alpha_n}\) and a permutation \(\sigma\) of \(\interval{n}\),
	the integral form permuted-basement Macdonald polynomial \(\mathcal{E}_\alpha^\sigma(\mathbf{x}; q, t)\) can be expressed as a sum one over all non-attacking augmented fillings using the signed alphabet \(\interval{n}_\pm\),
	\begin{equation} \label{eq:calE-NAF-signed}
		\mathcal{E}^{\sigma}_{\alpha}(\mathbf{x}; q, t) =
		\sum_{T \in \NAF[\interval{n}_\pm](\alpha, \sigma)}
		\mathbf{x}^{|T|}
		q^{\maj(T)} t^{\coinv(T)}
		(-t)^{\operatorname{neg}(T)}.
	\end{equation}
\end{proposition}

\begin{proof}
	Let \(S \in \NAF[\interval{n}_\pm](\mu, \sigma)\) be a non-attacking signed augmented filling, and let \(|S| \in \NAF[\interval{n}](\mu, \sigma)\) be the non-attacking unsigned augmented filling obtained from \(S\) by taking the absolute value of each entry.

	For any triple \((u, v, w)\), since \(u\) attacks \(v\) and \(v\) attacks \(w\), the non-attacking condition forces
	\(|S(u)| \neq |S(v)|\) and \(|S(v)| \neq |S(w)|\).
	Define the \emph{absolute coinversion number} \(\operatorname{acoinv}(S)\) to be the number of coinversion triples \((u, v, w)\) in \(S\) for which \(|S(u) \neq |S(w)|\) (and consequently \(|S(u)|\), \(|S(v)|\), and \(|S(w)|\) are all distinct).
	Define the \emph{signed coinversion number} \(\operatorname{scoinv}(S)\) to be the number of coinversion triples \((u, v, w)\) in \(S\) for which \(|S(u)| = |S(w)|\).
	Then, we have
	\begin{equation}
		\coinv(S) = \operatorname{acoinv}(S) + \operatorname{scoinv}(S).
	\end{equation}
	Note that, since \(\ind{a, b, a} = \ind{a, b} + \ind{b, a} - \ind{a, a} = 1\) for all distinct \(a, b \in \interval{n}_\pm\), we have \(\operatorname{scoinv}(|S|) = 0\) and consequently \(\operatorname{acoinv}(S) = \operatorname{acoinv}(|S|) = \coinv(|S|)\).

	Similarly, define the \emph{absolute major index} \(\operatorname{amaj}(S)\) and the \emph{signed major index} \(\operatorname{smaj}(S)\) by
	\begin{equation}
		\operatorname{amaj}(S) = \sum_{\substack{u \in \Des(S) \\ |S(u)| \neq |S(\south{u})|}} (1 + \leg(u)) 
		\quad \text{and} \quad
		\operatorname{smaj}(S) = \sum_{\substack{u \in \Des(S) \\ |S(u)| = |S(\south{u})|}} (1 + \leg(u)).
	\end{equation}
	Then, we have
	\begin{equation}
		\maj(S) = \operatorname{amaj}(S) + \operatorname{smaj}(S).
	\end{equation}
	Note that, since \(\ind{a, a} = 0\) for all \(a \in \interval{n}_\pm\), we have \(\operatorname{smaj}(|S|) = 0\) and consequently \(\operatorname{amaj}(S) = \operatorname{amaj}(|S|) = \maj(|S|)\).

	We group the augmented fillings \(S \in \NAF[\interval{n}_\pm](\mu, \sigma)\) by their absolute augmented filling \(|S|\) to rewrite the right-hand side of \Cref{eq:calE-NAF-signed} as
	\begin{equation*}
		\sum_{T \in \NAF[\interval{n}](\mu, \sigma)}
		\sum_{\substack{S \in {\NAF[\interval{n}_\pm](\mu, \sigma)} \\ |S| = T}}
		\mkern -10mu
		\mathbf{x}^{|S|} q^{\maj(S)} t^{\coinv(S)} (-t)^{\operatorname{neg}(S)}.
	\end{equation*}
	Fix a non-attacking unsigned augmented filling \(T \in \NAF[\interval{n}](\mu, \sigma)\).
	Using the decompositions of \(\maj(S)\) and \(\coinv(S)\) above,
	the inner sum corresponding to \(T\) can be rewritten as
	\begin{equation*}
		\sum_{\substack{S \in {\NAF[\interval{n}_\pm](\mu, \sigma)} \\ |S| = T}}
		\mkern -16mu
		\mathbf{x}^{|S|} q^{\maj(S)} t^{\coinv(S)} (-t)^{\operatorname{neg}(S)}
		= \mathbf{x}^T q^{\operatorname{maj}(T)} t^{\coinv(T)}
		\mkern -16mu
		\sum_{\substack{S \in {\NAF[\interval{n}_\pm](\mu, \sigma)} \\ |S| = T}}
		\mkern -16mu
		q^{\operatorname{smaj}(S)} t^{\operatorname{scoinv}(S)} (-t)^{\operatorname{neg}(S)}.
	\end{equation*}

	Let \(u \in \adg(\mu, \sigma)\) such that \(u\) is in row \(0\).
	Then, in any signed augmented filling \(S\) with \(|S| = T\),
	the box \(u\) does not contribute to \(\operatorname{smaj}(S)\) since there are no descents in row \(0\),
	neither to \(\operatorname{neg}(S)\) since the basement entries are entries of \(\sigma\) and thus positive.
	Moreover, there are no triples \((u, v, w)\) with \(u\) in row \(0\),
	so there is no contribution to \(\operatorname{scoinv}(S)\) from such boxes.

	Let \(u \in \dg(\mu)\) such that \(T(u) \neq T(\south{u})\).
	Then, in any signed augmented filling \(S\) with \(|S| = T\),
	the box \(u\) does not contribute to \(\operatorname{smaj}(S)\), and any triple \((u, v, w)\) does not contribute to \(\operatorname{scoinv}(S)\).
	The entry in box \(u\) contributes to \(\operatorname{neg}(S)\) if and only if \(u \in \interval{n}_-\).

	Now, let \(u \in \dg(\mu)\) such that \(T(u) = T(\south{u})\),
	and let \(S\) be a signed augmented filling with \(|S| = T\).
	The box \(u\) contributes to \(\operatorname{smaj}(S)\) if and only if \(\ind{S(u), S(\south{u})} = 1\),
	which occurs if and only if \(S(u) \in \interval{n}_-\).
	Moreover, any triple \((u, v, w)\) contributes to \(\operatorname{scoinv}(S)\) if and only if \(\ind{S(u), S(v), S(w)} = 0\),
	which occurs if and only if \(S(u) \in \interval{n}_-\).
	Recall that there are \(\arm(u)\) such triples \((u, v, w)\).
	Finally, the entry in box \(u\) contributes to \(\operatorname{neg}(S)\) if and only if \(S(u) \in \interval{n}_-\).

	Note that all contributions to \(\operatorname{smaj}(S)\), \(\operatorname{scoinv}(S)\), and \(\operatorname{neg}(S)\) are counted exactly once in the cases above,
	and their contributions only depend on the sign of the entry in box \(u\).
	Therefore, we can factor the sum over all \(S\) with \(|S| = T\) as a product over all boxes \(u \in \dg(\mu)\) (which excludes row \(0\), because there is no choice of sign and no contribution from those boxes).
	If \(T(u) \neq T(\south{u})\), the contribution from box \(u\) is
	\begin{equation}
		1 + (-t) = 1 - t,
	\end{equation}
	while if \(T(u) = T(\south{u})\), the contribution from box \(u\) is
	\begin{equation}
		1 + q^{1 + \leg(u)} t^{\arm(u)} (-t) = 1 - q^{1 + \leg(u)} t^{1 + \arm(u)}.
	\end{equation}
	Therefore, the inner sum corresponding to a fixed \(T \in \NAF[\interval{n}](\mu, \sigma)\) is
	\begin{equation*}
		\sum_{\substack{S \in {\NAF[\interval{n}_\pm](\mu, \sigma)} \\ |S| = T}}
		\mkern -10mu
		q^{\operatorname{smaj}(S)} t^{\operatorname{scoinv}(S)} (-t)^{\operatorname{neg}(S)}
		=
		\mkern -10mu
		\prod_{\substack{u \in \dg(\mu) \\ T(u) = T(\south{u})}}
		\mkern -18mu
		(1 - q^{1 + \leg(u)} t^{1 + \arm(u)})
		\mkern -10mu
		\prod_{\substack{u \in \dg(\mu) \\ T(u) \neq T(\south{u})}}
		\mkern -18mu
		(1 - t),
	\end{equation*}
	and adding over all \(T \in \NAF[\interval{n}](\mu, \sigma)\) gives the desired result.
\end{proof}

\begin{proposition} \label{prop:calE-Fill-signed}
	For a composition \(\alpha = \composition{\alpha_1, \ldots, \alpha_n}\) and a permutation \(\sigma\) of \(\interval{n}\),
	the integral form permuted-basement Macdonald polynomial \(\mathcal{E}_\alpha^\sigma(\mathbf{x}; q, t)\) can be expressed as a sum one over all augmented fillings using the signed alphabet \(\interval{n}_\pm\),
	\begin{equation} \label{eq:calE-Fill-signed}
		\mathcal{E}^{\sigma}_{\alpha}(\mathbf{x}; q, t) =
		\sum_{T \in \Fill[\interval{n}_\pm](\alpha, \sigma)}
		\mathbf{x}^{|T|}
		q^{\maj(T)} t^{\coinv(T)}
		(-t)^{\operatorname{neg}(T)}.
	\end{equation}
\end{proposition}

\begin{proof}
	We define a sign-reversing involution on the set of attacking augmented fillings in \(\Fill[\interval{n}_\pm](\alpha, \sigma) \setminus \NAF[\interval{n}_\pm](\alpha, \sigma)\).
	Recall that, in \(\interval{n}_\pm\), we have \( 1 < \bar{1} < 2 < \bar{2} < \cdots < n < \bar{n}\), which has the useful properties given in \Cref{eq:involution-property-1,eq:involution-property-2}.

	Given an attacking augmented filling \(T \in \Fill[\interval{n}_\pm](\alpha, \sigma) \setminus \NAF[\interval{n}_\pm](\alpha, \sigma)\),
	let \(a\) be the smallest label which is the absolute value of two attacking entries in \(T\), which exists since \(T\) is attacking.
	Let \(v\) be the earlier box (in reading order) with entry of absolute value \(a\) that is attacked by another box with entry of absolute value \(a\).
	Let \(u\) be the earlier box (in reading order) which attacks \(v\) and has absolute value \(a\).
	Since the entries in row \(0\) are the basement entries given by \(\sigma\), which are all distinct and positive, we have that \(u\) is not in row \(0\).
	Define \(T'\) to be the filling obtained from \(T\) by switching the sign of the entry in box \(u\).
	Since \(u\) is not in row \(0\), \(T'\) has basement \(\sigma\).
	Then, \(T'\) is an attacking augmented filling in \(\Fill[\interval{n}_\pm](\alpha, \sigma) \setminus \NAF[\interval{n}_\pm](\alpha, \sigma)\).

	Since the definitions of \(a\), \(v\) and \(u\) only depend on the absolute values of the entries in \(T\),
	we have that applying the same procedure to \(T'\) returns \(T\).
	Hence, the map \(T \mapsto T'\) is an involution.

	Without loss of generality, assume that \(T(u) = a\) and \(T'(u) = \bar{a}\).
	We have \(\mathbf{x}^{|T'|} = \mathbf{x}^{|T|}\) and \(\operatorname{neg}(T') = \operatorname{neg}(T) + 1\).

	I claim that \(\Des(T') = \Des(T)\), which implies that \(\maj(T') = \maj(T)\).
	Let \(w\) be the box directly above \(u\), if it exists.
	For any box \(b \notin \{w, u\}\), the fillings \(T\) and \(T'\) agree on the entries in boxes \(b\) and \(\south{b}\),
	so \(b\) is a descent in \(T\) if and only if it is a descent in \(T'\).
	For \(w\), if it exists, we can apply \Cref{eq:involution-property-1} to conclude that \(w\) is a descent in \(T\) if and only if it is a descent in \(T'\).
	For \(u\), note that \( d(u) \) is attacked by \(v\) and is earlier than \(v\) in reading order.
	Therefore, by minimality of \(v\), we have \(|T(d(u))| \neq a\), so we can apply \Cref{eq:involution-property-2} to conclude that \(u\) is a descent in \(T\) if and only if it is a descent in \(T'\).
	Thus, we have shown that \(\Des(T') = \Des(T)\), which implies that \(\maj(T') = \maj(T)\).

	I claim that \(\inv(T') = \inv(T) + 1\).
	Using \Cref{prop:inv-via-inversion-pairs-augmented}, since \(\Des(T') = \Des(T)\), it suffices to show that \(\left|\InversionPairs(T')\right| = \left|\InversionPairs(T)\right| + 1\).
	Any pair of attacking boxes that does not involve box \(u\) is an inversion pair in \(T\) if and only if it is an inversion pair in \(T'\).
	Moreover, by \Cref{eq:involution-property-1}, any box \(b\) that attacks \(u\) forms an inversion pair \((b, u)\) in \(T\) if and only if it forms an inversion pair \((b, u)\) in \(T'\).
	Consider a box \(b\) that is attacked by \(u\).
	If \(b = v\), then \((u, v)\) is an inversion pair in \(T'\) but not in \(T\).
	If \(b \neq v\), then by minimality of \(u\), we have \(|T(b)| \neq a\), so by \Cref{eq:involution-property-2}, \((u, b)\) is an inversion pair in \(T\) if and only if it is an inversion pair in \(T'\).
	Therefore, \(\InversionPairs(T')\) has exactly one more inversion pair than \(\InversionPairs(T)\), namely \((u, v)\),
	which proves the claim.
	Consequently, \(\coinv(T') = \coinv(T) - 1\).

	Putting this together, we have
	\begin{align*}
		\mathbf{x}^{|T'|} q^{\maj(T')} t^{\coinv(T')} (-t)^{\operatorname{neg}(T')}
		&= \mathbf{x}^{|T|} q^{\maj(T)} t^{\coinv(T) - 1} (-t)^{\operatorname{neg}(T) + 1} \\
		&= - \mathbf{x}^{|T|} q^{\maj(T)} t^{\coinv(T)} (-t)^{\operatorname{neg}(T)},
	\end{align*}
	which implies that
	\begin{equation*}
		\sum_{T \in \Fill[\interval{n}_\pm](\alpha, \sigma) \setminus \NAF[\interval{n}_\pm](\alpha, \sigma)}
		\mathbf{x}^{|T|} q^{\maj(T)} t^{\coinv(T)} (-t)^{\operatorname{neg}(T)} = 0.
	\end{equation*}
	and consequently
	\begin{equation*}
		\sum_{T \in \Fill[\interval{n}_\pm](\alpha, \sigma)}
		\mathbf{x}^{|T|} q^{\maj(T)} t^{\coinv(T)} (-t)^{\operatorname{neg}(T)}
		=
		\sum_{T \in \NAF[\interval{n}_\pm](\alpha, \sigma)}
		\mathbf{x}^{|T|} q^{\maj(T)} t^{\coinv(T)} (-t)^{\operatorname{neg}(T)}.
	\end{equation*}
	The right-hand side equals \(\mathcal{E}^{\sigma}_{\alpha}(\mathbf{x}; q, t)\) by \Cref{prop:calE-NAF-signed},
	so the result follows.
\end{proof}
