\chapter{Bijective combinatorics for Macdonald polynomials} \label{chap:bijective}

In this chapter, we describe \cite{KLO22}'s bijection between
\begin{equation*}
    \Fill[\mathcal{A}](\mu) \quad \text{and} \quad \Fill[\mathcal{A}](s_i \permact \mu),
\end{equation*}
which preserves the statistics $\inv$ and $\maj$.

The existence of this bijection implies that 
\begin{equation}
    \tilde{H}_\mu[X;q,t] = \tilde{H}_{s_i \permact \mu}[X;q,t],
\end{equation}
and by applying this result repeatedly, we obtain
\begin{equation}
    \tilde{H}_{\mu}[X;q,t] = \tilde{H}_{\alpha}[X;q,t],
\end{equation}
where $\alpha$ is a rearrangement of the partition $\mu$.

\section{The bijection}

In this section, we define the bijection \(\phi \colon \Fill(\alpha) \to \Fill(s_i \alpha)\) from \cite{KLO22} that preserves the statistics \(\inv\) and \(\maj\).
As an auxiliary construction,
we also define another bijection \(\psi \colon \Fill(s_i \alpha) \to \Fill(\alpha)\) that changes the statistics \(\inv\) and \(\maj\) in a controlled way.
The manner in which \(\psi\) changes the statistics is specified in \Cref{thm:...}.
This auxiliary bijection \(\psi\) will also be important in a later section.

Without loss of generality, we restrict to the case \(\alpha_{i} > \alpha_{i+1}\), as the case \(\alpha_i < \alpha_{i+1}\) is symmetric.

Both of \(\phi\) and \(\psi\) follow a very rigid structure.
If a filling \(T \in \Fill(\alpha)\) is mapped to a filling \(U \in \Fill(s_i \alpha)\) by one of these maps, then
\begin{itemize}
    \item \(T(j, r) = U(j, r)\) for all \(j \in \interval{n} \setminus \{i, i+1\}\) and all \(r \in \interval{\alpha_j}\),
    that is, all columns except \(i\) and \(i+1\) are identical in \(T\) and \(U\);
    \item \(T(i, r) = U(i+1, r)\) for all \(r \in \interval{\alpha_{i+1} + 1, \alpha_i}\),
    that is, the topmost \(\alpha_i - \alpha_{i+1}\) entries in column \(i\) of \(T\) are copied to the topmost \(\alpha_i - \alpha_{i+1}\) entries in column \(i+1\) of \(U\);
    \item \(\{ T(i, r), T(i+1, r) \} = \{ U(i, r), U(i+1, r) \}\) for all \(r \in \interval{\alpha_{i+1}}\),
    that is, the entries in rows \(1, \dots, \alpha_{i+1}\) of columns \(i\) and \(i+1\) in \(T\) and \(U\) are the same, but they might be in the same positions or in swapped positions.
\end{itemize}

Given these restrictions, the only freedom we have in defining \(\phi\) and \(\psi\) is to decide, for each \(r \in \interval{\alpha_{i+1}}\), whether to swap the entries in row \(r\) of columns \(i\) and \(i+1\) or not.
In other words, the only freedom we have is to decide, for each \(r \in \interval{\alpha_{i+1}}\), whether to set
\begin{equation*}
    (U(i,r), U(i+1,r)) = (T(i,r), T(i+1,r)) \quad \text{or} \quad (U(i,r), U(i+1,r)) = (T(i+1,r), T(i,r)).
\end{equation*}

Let \(T \in \Fill(\alpha)\).
Let \(i \in [n-1]\) be such that \(\alpha_i > \alpha_{i+1}\).
If you want to compute \(U = \phi(T)\), assign a coin value of \(\phi \in \{\phi,\psi\}\) to row \(1\); if you want to compute \(U = \psi(T)\), assign a coin value of \(\psi \in \{\phi,\psi\}\) to row \(1\).

For \(r\) ranging from \(1\) to \(\alpha_i\), do the following:

Assume \(r \neq \alpha_{i+1}\).
Let \(a = T(i,r+1)\), \(b = T(i+1,r+1)\), \(c = T(i,r)\), and \(d = T(i+1,r)\).  

\textbf{Case I:} If \(\ind{a, c} + \ind{b, c} - \ind{a, d} - \ind{b, d} \neq 0\) (cases 3, 4, 9, 10 in \cite{KLO22}),
assign the temporary label ``C'' to row \(r\) and  
assign the coin value of row \(r+1\) as the same coin value as row \(r\) if \(\chi_{ab} = \chi_{cd}\) (3,9), and as the opposite coin value as row \(r\) if \(\chi_{ab} \neq \chi_{cd}\) (4,10).

\textbf{Case II:} If \(\ind{a, c} = \ind{a, d} = \ind{b, c} = \ind{b, d}\) (cases 1, 2, 11, 12 in \cite{KLO22}):
assign the label ``N'' (resp.\ ``S'') to row \(r\) if the coin value of row \(r\) is \(\phi\) (resp.\ \(\psi\)), and  
assign the coin value \(\phi\) to row \(r+1\).

\textbf{Case III:} If \(\ind{a, c} = \ind{b, c} \neq \ind{a, d} = \ind{b, d}\) (cases 7, 8 in \cite{KLO22}):
assign the label ``S'' (resp.\ ``N'') to row \(r\) if the coin value of row \(r\) is \(\phi\) (resp.\ \(\psi\)), and  
assign the coin value of \(\phi\) to row \(r+1\).

\textbf{Case IV:} If \(\ind{a, c} = \ind{a, d} \neq \ind{b, c} = \ind{b, d}\) (cases 5, 6 in \cite{KLO22}):
assign the label ``N'' (resp.\ ``S'') to row \(r\) if the coin value of row \(r\) is \(\phi\) (resp.\ \(\psi\)), and
assign the coin value of \(\psi\) to row \(r+1\).

Assume \(r = \alpha_{i+1}\).
Let \(a = T(i,r+1)\), \(c = T(i,r)\), and \(d = T(i+1,r)\).

\textbf{Base case I:} If \(\ind{a, c} \neq \ind{a, d}\), assign the label ``S'' (resp.\ ``N'') to row \(r\) if the coin value of row \(r\) is \(\phi\) (resp.\ \(\psi\)).

\textbf{Base case II:} If \(\ind{a, c} = \ind{a, d}\), assign the label ``N'' (resp.\ ``S'') to row \(r\) if the coin value of row \(r\) is \(\phi\) (resp.\ \(\psi\)).

Now, for each row temporarily labeled ``C'', replace its label with the label of the closest row above (i.e., with greater row index) that is not labeled ``C''.
Finally,
for a row \(r\) with label ``N'', set \(U(i,r) = T(i,r)\) and \(U(i+1,r) = T(i+1,r)\), and
for a row \(r\) with label ``S'', set \(U(i,r) = T(i+1,r)\) and \(U(i+1,r) = T(i,r)\).
